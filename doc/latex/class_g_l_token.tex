\doxysubsection{Detailed Description}
Token des Vier Gewinnt Spiels. 

Wird abgeleitetet durch die Klassen \mbox{\hyperlink{class_g_l_token_red}{GLToken\+Red}} und \mbox{\hyperlink{class_g_l_token_green}{GLToken\+Green}}.

Das Attribut m\+\_\+rotated gibt an, ob das Token rotiert wurde.

Über das Attribut m\+\_\+movable kann gestuert werden, ob das Token bewegbar ist.
