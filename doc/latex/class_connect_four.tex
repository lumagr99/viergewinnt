\doxysubsection{Detailed Description}
Die Connect\+Four-\/\+Klasse enthält die Spiellogik für ein Vier Gewinnt Spiel. Die Klasse enthält die Tischplatte, das Spielbrett und die 42 Spieltoken. 

Die draw-\/\+Methode zeichnet die enthaltenen Objekte und benötigt einen aktiven Open\+GL-\/\+Kontext. Über die Methode select\+Token kann ein Token ausgewählt und anschließend mit der Methode move\+Token bewegt werden. Mithilfe der Methode deselect\+Token kann das ausgewählte Token deselektiert werden.

Das Spiel startet mit dem im Konstruktor übergebenen Spieler. Bei Spielende wird ein game\+Over Signal mit Gewinnerfarbe emittiert.

Bei folgenden Aktionen wird ein sound\+Requested Signal mit dem Dateinamen der abzuspielenden Datei emittiert\+:
\begin{DoxyItemize}
\item Bei Selektion eines Tokens
\item Beim Einfügen eines Tokens in das Spielbrett
\item Beim Spielerwechsel
\item Beim Spielende 
\end{DoxyItemize}
