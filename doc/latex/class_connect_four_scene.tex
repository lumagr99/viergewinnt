\doxysubsection{Detailed Description}
Die Connect\+Four\+Scene-\/\+Klasse ist abgeleitet von \mbox{\hyperlink{class_g_l_item}{GLItem}}. Die Klasse verarbeitet Mauseingaben für das Vier Gewinnt Spiel. Die Eingaben werden in m\+\_\+mouse\+Near, m\+\_\+mouse\+Far und m\+\_\+last\+Intersection gespeichert.
